# 9beads-acme

Acme frontend for beads task tracking.

## Usage

```sh
mk install
Beads
```

Requires 9beads server running.

## Features

- Task list with filters (All, Ready, Deferred)
- Create, edit, delete beads
- Mount/unmount project directories
- Comments support
- Inline edit commands (-, c, u, o, d, x, f, +)
